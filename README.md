# CyberSecurity Awareness Bot – PROG POE Part 2 (2026)

## Project Overview

The **CyberSecurity Awareness Bot** is a more advanced version of the Part 1 chatbot application developed in **C# using .NET 8.0**. This project focuses on improving user interaction, chatbot intelligence, and cybersecurity education through enhanced functionality and structured programming techniques.
The chatbot allows users to interact through a console-based interface where they can learn about cybersecurity threats, safe online practices, and digital protection strategies in an engaging way.

## Purpose of the Project
The purpose of this project is to:
- Educate users about cybersecurity awareness
- Demonstrate practical software development skills
- Improve chatbot interaction and responsiveness
- Apply object-oriented programming principles
- Create a user-friendly educational application
- Expand upon the functionality developed in Part 1
## Features
### Interactive Cybersecurity Chatbot
The chatbot communicates with users through a conversational interface.

### Personalized Experience
The application stores and uses the user’s name during interactions.

### Keyword Recognition
The chatbot can detect cybersecurity-related keywords and respond appropriately.

### Multiple Cybersecurity Topics
Users can learn about:
- Phishing
- Password safety
- Malware
- Social engineering
- Safe browsing
- Online scams
- Data protection

### Typing Animation Effect
Responses are displayed gradually to simulate realistic conversation.

### Audio Greeting
A welcome sound is played when the application starts.

### Color-Coded Console Interface
Different console colors improve readability and user experience.

### Error Handling
The application validates user input and prevents crashes caused by invalid entries.

### Modular Programming Structure
The project is organized into methods and classes for improved maintainability and scalability.
